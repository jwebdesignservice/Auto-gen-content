# Fast Launch — Video Log & Experiment Tracker

Purpose: log **every** video we publish with the creative variables we changed, so
we can deliberately mix things up (content type, images/projects, heading copy,
accent colour, background, hook style) and then correlate against performance to
find what actually works. Fill in the performance columns once a post has had time
to land (check back ~48-72h, then again at a week).

**How to use:** add a new row per video at the top of the table. Keep variables
distinct between consecutive posts so comparisons are meaningful. When something
clearly out-performs, note the takeaway in "Learnings / next test".

## Variables we can rotate to test

- **Content type:** work-showcase · single case-study · educational/tips · before-after teardown
- **Hook heading copy:** "I Design & Build Websites" · "Building websites is easy…" · "We build websites" · "5 things every site needs"
- **Accent colour:** orange (brand) · green · blue · per-project
- **Background:** warm gradient · cool gradient · solid · subtle pattern
- **Cover images:** light hero vs dark hero · 1 vs 2 vs 3 mockups · which industries
- **Projects featured:** which mix, and the order
- **Caption thesis / CTA wording**
